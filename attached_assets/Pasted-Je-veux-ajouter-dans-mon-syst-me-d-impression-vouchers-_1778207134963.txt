Je veux ajouter dans mon système d’impression vouchers un réglage d’échelle EXACTEMENT comme Microsoft Edge pendant l’impression.

OBJECTIF :
Pouvoir changer dynamiquement l’échelle d’impression :
- 100%
- 95%
- 90%
- 85%
- 80%
- etc.

comme dans Edge.

IMPORTANT :
Le changement doit modifier :
- taille des tickets
- densité
- QR
- polices
- espacements
- nombre de vouchers par page

exactement comme l’échelle native d’impression Edge.

FONCTIONNEMENT DEMANDÉ :

Ajouter un sélecteur :

<select id="printScale">
  <option value="1">100%</option>
  <option value="0.95">95%</option>
  <option value="0.90">90%</option>
  <option value="0.85" selected>85%</option>
  <option value="0.80">80%</option>
</select>

Quand la valeur change :
appliquer automatiquement le scale à toute la page d’impression.

IMPORTANT :
Le scale doit agir EXACTEMENT comme Edge print scale.

UTILISER CETTE MÉTHODE :

body.scaled {
  transform-origin: top left;
}

Puis en JavaScript :

function applyPrintScale(scale){

  const body = document.body;

  body.style.transform = `scale(${scale})`;

  body.style.width = `${100 / scale}%`;

}

document
.getElementById('printScale')
.addEventListener('change', function(){

   applyPrintScale(this.value);

});

window.addEventListener('load', () => {

   applyPrintScale(0.85);

});
