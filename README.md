# nanoTECH VoucherProd (VoucherNet)

Monorepo **pnpm** : API Express + front Vite + packages partagés (Drizzle, OpenAPI, etc.).

## Démarrer en local

Voir **[DEVELOPMENT.md](./DEVELOPMENT.md)** (PostgreSQL, `DATABASE_URL`, commandes `pnpm`, mobile Expo, dépannage Windows).

**Raccourci** (Docker + `.env` avec Neon) : `pnpm run dev:full` — synchro Neon → Postgres local, écriture de `.env.local`, puis API + Vite.
